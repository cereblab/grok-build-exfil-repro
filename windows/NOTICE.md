[`cereblab/grok-build-exfil-repro`](https://github.com/cereblab/grok-build-exfil-repro).
The imported upstream history is retained in this Git repository. The refactor
started from upstream commit `4217f340dd10384436105191809c1af1f58b08ee`,
authored by Cereblab.

software license. There is therefore no upstream license text to reproduce.

upstream author did not state. Consult the upstream project and its history for
